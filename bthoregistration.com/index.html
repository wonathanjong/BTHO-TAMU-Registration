bthoRegistration
bthoRegistration is a tool that optimizes class scheduling for TAMU students.

Why Use It?
  1. Grade Optimization
    Class Sections are chosen based on previous grade distributions for different professors.
  2. User Preferences
    Class Sections can be chosen based on user preferences, such as preferred class times and professors.
  3. The Future
    This extension automatically updates as classes and professors change each semester.
